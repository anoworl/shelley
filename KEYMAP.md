> **Note:** Shortcuts are disabled when focus is in `input` or `textarea` elements.

## Global

| Key | Description |
| --- | ----------- |
| `Ctrl+Enter` | Send message |
| `Shift+Enter` | Insert newline |

## Multi-pane Mode

Active when grid layout is 2x1 or larger (not 1x1).

### Pane Navigation

| Key | Description |
| --- | ----------- |
| `Tab` | Next pane (cycles) |
| `Shift+Tab` | Previous pane (cycles) |
| `Alt+←` | Move to left pane |
| `Alt+→` | Move to right pane |
| `Alt+↑` | Move to upper pane |
| `Alt+↓` | Move to lower pane |
| `h` | Move left (wraps to previous row) |
| `l` | Move right (wraps to next row) |

### Scrolling

| Key | Description |
| --- | ----------- |
| `j` | Scroll down |
| `k` | Scroll up |
| `g` `g` | Scroll to top |
| `G` | Scroll to bottom |

### Actions

| Key | Description |
| --- | ----------- |
| `Enter`, `Space` | Open input modal |
| `Escape` | Close input modal |
| `n` | New conversation |
| `q` | Close pane |

### Quick Reply

| Key | Description |
| --- | ----------- |
| `s` | Stop agent (when working) |
| `y` | Send "yes" |
| `c` | Send "continue" |

## Input Field

| Key | Description |
| --- | ----------- |
| `Enter` | Send (behavior depends on settings) |
| `Ctrl+Enter` | Send (always) |
| `Shift+Enter` | Insert newline |
